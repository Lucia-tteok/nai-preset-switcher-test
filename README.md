# NAI预设一键切换 v1.3.8

一个用于 NAI（NovelAI）预设一键切换的脚本，配合智绘姬（st-chatu8）使用。

## 功能

- 预设分类管理（古风、西幻、现代、科幻、二次元、写实、未分类）
- 收藏预设的正向 / 负向提示词一键应用
- 面板化的预设浏览与切换界面

## 安装方式

### 方式一：作为扩展安装（Git URL）

在酒馆「扩展 → 安装扩展」中填入本仓库地址安装：

```
https://github.com/Lucia-tteok/nai-preset-switcher
```

> 注意：本脚本原为「酒馆助手（JS-Slash-Runner）」环境编写，做成独立扩展后需实测兼容性。

### 方式二：作为脚本导入（推荐）

在「酒馆助手」扩展的「脚本库」中导入 `NAI预设一键切换 v1.3.8.json` 即可。

## 文件说明

- `NAI预设一键切换 v1.3.8.json` —— 酒馆助手脚本库导入格式
- `index.js` —— 提取出的脚本主体（供扩展方式加载）
- `manifest.json` —— 扩展清单文件
