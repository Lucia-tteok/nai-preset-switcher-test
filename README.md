# NAI切换助手 v1.4.2

一个配合智绘姬（st-chatu8）使用的 NovelAI 预设管理与双击切换扩展。

它不只是“预设切换”，还包含预设自动分类、收藏管理、Vibe 分组、随机抽卡、日夜主题、拖拽排序和自动更新检测等面板功能。

## 主要功能

- 预设库管理：浏览、收藏并管理 NAI 预设。
- 双击应用预设：双击预设卡片即可应用对应正向 / 负向提示词。
- 自动分类：根据预设内容自动归类到古风、西幻、现代、科幻、二次元、写实、未分类等类别。
- 拖拽排序：长按预设卡片可调整显示顺序。
- 随机抽取：从预设卡片中随机抽取一个预设。
- Vibe 分组管理：读取智绘姬中的 Vibe 数据，并支持创建、重命名、删除、保存 Vibe 叠加组。
- 日夜切换：支持面板明暗主题切换。
- 自动检测更新：插件会自动检测新版本，并可在更新中心查看版本与更新内容。

## 安装方式

### 方式一：作为扩展安装

在酒馆中打开：

```text
扩展 → 安装扩展
```

填入本仓库地址：

```text
https://github.com/Lucia-tteok/nai-preset-switcher
```

安装完成后刷新页面即可。

### 方式二：作为脚本导入

如果你使用的是「酒馆助手（JS-Slash-Runner）」脚本库，可以导入仓库内的脚本 JSON 文件：
```text
NAI切换助手 v1.4.2.json
```


## 使用教程

### 1. 打开面板

安装并刷新页面后，在酒馆魔棒菜单界面中打开 `NAI切换助手` 面板。

### 2. 管理和应用预设

1. 在预设库中浏览已有 NAI 预设。
2. 插件会根据预设内容自动分类，方便快速找到需要的风格。
3. 双击预设卡片即可将对应正向 / 负向提示词应用到智绘姬。
4. 对常用预设进行收藏，方便下次快速切换。

### 3. 调整预设顺序

长按预设卡片并拖动，可以改变预设卡片在面板中的排列顺序。

适合把常用预设放到更靠前的位置。

### 4. 随机抽取预设

在面板中使用随机抽取功能，可以从当前预设卡片中随机选择一个预设。

适合不知道用哪个风格时快速尝试。

### 5. 使用 Vibe 分组

插件会读取智绘姬中已经存在的 Vibe 数据。

你可以在 Vibe 页面中：

- 查看已有 Vibe。
- 新建 Vibe 叠加组。
- 将 Vibe 加入当前组。
- 调整组内 Vibe 强度。
- 保存、重命名或删除 Vibe 组。

注意：当前版本不提供 Vibe 导入入口。Vibe 文件请先在智绘姬中导入，再回到本插件同步和分组使用。

### 6. 切换日夜主题

在面板中使用日夜切换按钮，可以切换浅色 / 深色显示效果。

### 7. 自动检测更新

插件会自动检测是否存在新版本。

如果检测到更新，可以点击面板标题旁的版本号或提示按钮打开更新中心，查看当前版本、最新版本和更新内容。更新后刷新页面即可生效。

## 文件说明

- `manifest.json`：扩展清单，包含扩展名称、版本、作者和更新说明。
- `index.js`：扩展主体代码。
- `NAI切换助手 v1.4.2.json`：酒馆助手脚本库导入格式。

## 注意事项

- 本插件需要配合智绘姬（st-chatu8）使用。
- Vibe 数据来源于智绘姬，本插件主要负责读取、分组和应用。
- 如果更新后界面没有变化，请刷新页面或重新安装扩展。
